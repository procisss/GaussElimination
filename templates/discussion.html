Mathematical Discussion

Introduction

Gaussian Elimination is an algorithm used to solve systems of linear equations by transforming an augmented matrix into upper triangular form using elementary row operations. Once in triangular form, the system can be solved efficiently through back substitution.

The method is widely used in engineering, physics, computer science, and numerical analysis because it provides a systematic way to solve multiple equations with multiple unknown variables.

What is Gaussian Elimination?

Gaussian Elimination simplifies a system of equations step-by-step by eliminating variables below each pivot element. The process uses elementary row operations to preserve the solution while converting the matrix into an upper triangular form.

After elimination is complete, the remaining triangular system is solved using back substitution.

Partial Pivoting

Partial pivoting improves numerical stability by swapping rows so that the largest absolute value in the current column becomes the pivot. This minimizes rounding errors and helps avoid division by very small or zero values.

Without pivoting, the elimination process may become unstable or fail for certain systems of equations.

Row Elimination Formula

mji = aji / aii
Rj ← Rj − mjiRi

The multiplier mji is calculated by dividing the element being eliminated by the pivot element. This operation removes coefficients below the pivot position.

Back Substitution

Once the matrix is in upper triangular form, the final equation is solved first. The resulting value is substituted into the equations above it step-by-step until all unknown variables are determined.

Back substitution works efficiently because each equation contains fewer unknowns than the previous one.

Advantages

  • Avoids division by zero through pivoting
  • Improves numerical stability in computations
  • Reduces rounding errors during elimination
  • Provides a systematic method for solving linear systems
  • Works efficiently for many engineering and scientific applications

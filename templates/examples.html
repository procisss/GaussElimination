Worked Examples

Step-by-step solutions using Gaussian Elimination with Partial Pivoting

Given System

x + y + z = 6

2x − y + z = 3

x + 2y − z = 3

Step 1: Augmented Matrix

[ 1 1 1 | 6 ]
[ 2 -1 1 | 3 ]
[ 1 2 -1 | 3 ]

Step 2: Partial Pivoting

Compare absolute values in column 1: |1|, |2|, and |1|. Largest is |2|, so swap Row 1 and Row 2.

R₁ ↔ R₂
[ 2 -1 1 | 3 ]
[ 1 1 1 | 6 ]
[ 1 2 -1 | 3 ]

Step 3: Eliminate Entries Below Pivot

Use the pivot value 2 in Row 1 to eliminate the x-values below it.

m₂₁ = 1/2

Multiply Row 1 by 1/2 and subtract it from Row 2.

R₂ → R₂ − (1/2)R₁

m₃₁ = 1/2

Multiply Row 1 by 1/2 and subtract it from Row 3.

R₃ → R₃ − (1/2)R₁
[ 2 -1 1 | 3 ]
[ 0 3/2 1/2 | 9/2 ]
[ 0 5/2 -3/2 | 3/2 ]

Step 4: Partial Pivoting (Column 2)

Compare absolute values in column 2: |3/2| and |5/2|. Largest is |5/2|, so swap Row 2 and Row 3.

R₂ ↔ R₃
[ 2 -1 1 | 3 ]
[ 0 5/2 -3/2 | 3/2 ]
[ 0 3/2 1/2 | 9/2 ]

Step 5: Eliminate Below Second Pivot

Use the pivot value 5/2 in Row 2 to eliminate the value below it in column 2.

m₃₂ = (3/2)/(5/2) = 3/5

Multiply Row 2 by 3/5 and subtract it from Row 3.

R₃ → R₃ − (3/5)R₂
[ 2 -1 1 | 3 ]
[ 0 5/2 -3/2 | 3/2 ]
[ 0 0 7/5 | 18/5 ]

Step 6: Back Substitution

Start from the bottom row and solve upward.

From Row 3:

(7/5)z = 18/5

z = (18/5)/(7/5) = 18/7

Substitute z into Row 2:

(5/2)y − (3/2)(18/7) = 3/2

y = 15/7

Substitute y and z into Row 1:

2x − (15/7) + (18/7) = 3

x = 9/7

Final Answer

x = 9/7

y = 15/7

z = 18/7

Given System

2x + y − z = 8

−3x − y + 2z = −11

−2x + y + 2z = −3

Step 1: Augmented Matrix

[ 2 1 -1 | 8 ]
[ -3 -1 2 | -11 ]
[ -2 1 2 | -3 ]

Step 2: Partial Pivoting

Compare absolute values in column 1: |2|, |-3|, and |-2|. Largest is |-3|, so swap Row 1 and Row 2.

R₁ ↔ R₂
[ -3 -1 2 | -11 ]
[ 2 1 -1 | 8 ]
[ -2 1 2 | -3 ]

Step 3: Eliminate Entries Below Pivot

Use the pivot value -3 in Row 1 to eliminate the x-values below it.

m₂₁ = 2 / (-3) = -2/3

Multiply Row 1 by -2/3 and subtract it from Row 2.

R₂ → R₂ − (-2/3)R₁

m₃₁ = (-2) / (-3) = 2/3

Multiply Row 1 by 2/3 and subtract it from Row 3.

R₃ → R₃ − (2/3)R₁
[ -3 -1 2 | -11 ]
[ 0 1/3 1/3 | 2/3 ]
[ 0 5/3 2/3 | 13/3 ]

Step 4: Partial Pivoting (Column 2)

Compare absolute values in column 2: |1/3| and |5/3|. Largest is |5/3|, so swap Row 2 and Row 3.

R₂ ↔ R₃
[ -3 -1 2 | -11 ]
[ 0 5/3 2/3 | 13/3 ]
[ 0 1/3 1/3 | 2/3 ]

Step 5: Eliminate Below Second Pivot

Use the pivot value 5/3 in Row 2 to eliminate the value below it in column 2.

m₃₂ = (1/3)/(5/3) = 1/5

Multiply Row 2 by 1/5 and subtract it from Row 3.

R₃ → R₃ − (1/5)R₂
[ -3 -1 2 | -11 ]
[ 0 5/3 2/3 | 13/3 ]
[ 0 0 1/5 | -1/5 ]

Step 6: Back Substitution

Start from the bottom row and solve upward.

From Row 3:

(1/5)z = -1/5

z = (-1/5)/(1/5) = -1

Substitute z into Row 2:

(5/3)y + (2/3)(-1) = 13/3

y = 3

Substitute y and z into Row 1:

-3x - 3 + 2(-1) = -11

x = 2

Final Answer

x = 2

y = 3

z = -1

Given System

x + y + z + w = 10

2x − y + z + 3w = 5

x + 2y − z + w = 7

3x + y + 2z − w = 12

Step 1: Augmented Matrix

[ 1 1 1 1 | 10 ]
[ 2 -1 1 3 | 5 ]
[ 1 2 -1 1 | 7 ]
[ 3 1 2 -1 | 12 ]

Step 2: Partial Pivoting

Compare absolute values in column 1: |1|, |2|, |1|, and |3|. Largest is |3|, so swap Row 1 and Row 4.

R₁ ↔ R₄
[ 3 1 2 -1 | 12 ]
[ 2 -1 1 3 | 5 ]
[ 1 2 -1 1 | 7 ]
[ 1 1 1 1 | 10 ]

Step 3: Eliminate Entries Below Pivot

Use the pivot value 3 in Row 1 to eliminate the x-values below it.

m₂₁ = 2/3

R₂ → R₂ − (2/3)R₁

m₃₁ = 1/3

R₃ → R₃ − (1/3)R₁

m₄₁ = 1/3

R₄ → R₄ − (1/3)R₁
[ 3 1 2 -1 | 12 ]
[ 0 -5/3 -1/3 11/3 | -3 ]
[ 0 5/3 -5/3 4/3 | 3 ]
[ 0 2/3 1/3 4/3 | 6 ]

Step 4: Partial Pivoting (Column 2)

Compare absolute values in column 2: |-5/3|, |5/3|, and |2/3|. Choose 5/3 as the pivot.

R₂ ↔ R₃
[ 3 1 2 -1 | 12 ]
[ 0 5/3 -5/3 4/3 | 3 ]
[ 0 -5/3 -1/3 11/3 | -3 ]
[ 0 2/3 1/3 4/3 | 6 ]

Step 5: Continue Elimination

Eliminate the y-values below the second pivot.

m₃₂ = (-5/3)/(5/3) = -1

R₃ → R₃ − (-1)R₂

m₄₂ = (2/3)/(5/3) = 2/5

R₄ → R₄ − (2/5)R₂
[ 3 1 2 -1 | 12 ]
[ 0 5/3 -5/3 4/3 | 3 ]
[ 0 0 -2 5 | 0 ]
[ 0 0 1 4/5 | 24/5 ]

Step 6: Back Substitution

Solve from the bottom row upward.

z + (4/5)w = 24/5

-2z + 5w = 0

Solving gives:

w = 2

z = 16/5

y = 18/5

x = -14/5

Final Answer

x = -14/5

y = 18/5

z = 16/5

w = 2

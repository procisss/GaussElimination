Worked Examples

Step-by-step solutions using Gaussian Elimination with Partial Pivoting

Given System

x + y + z = 6

2x − y + z = 3

x + 2y − z = 3

Step 1: Augmented Matrix

[ 1 1 1 | 6 ]
[ 2 -1 1 | 3 ]
[ 1 2 -1 | 3 ]

Step 2: Partial Pivoting

Compare absolute values in column 1: |1|, |2|, and |1|. Largest is |2|, so swap Row 1 and Row 2.

R₁ ↔ R₂
[ 2 -1 1 | 3 ]
[ 1 1 1 | 6 ]
[ 1 2 -1 | 3 ]

Step 3: Eliminate Entries Below Pivot

m₂₁ = 1/2

R₂ → R₂ − (1/2)R₁

m₃₁ = 1/2

R₃ → R₃ − (1/2)R₁
[ 2 -1 1 | 3 ]
[ 0 3/2 1/2 | 9/2 ]
[ 0 5/2 -3/2 | 3/2 ]

Step 4: Partial Pivoting (Column 2)

Compare absolute values in column 2: |3/2| and |5/2|. Largest is |5/2|, so swap Row 2 and Row 3.

R₂ ↔ R₃
[ 2 -1 1 | 3 ]
[ 0 5/2 -3/2 | 3/2 ]
[ 0 3/2 1/2 | 9/2 ]

Step 5: Eliminate Below Second Pivot

m₃₂ = (3/2)/(5/2) = 3/5

R₃ → R₃ − (3/5)R₂
[ 2 -1 1 | 3 ]
[ 0 5/2 -3/2 | 3/2 ]
[ 0 0 7/5 | 18/5 ]

Step 6: Back Substitution

z = (18/5)/(7/5) = 18/7

(5/2)y − (3/2)(18/7) = 3/2

y = 15/7

2x − (15/7) + (18/7) = 3

x = 9/7

Final Answer

x = 9/7

y = 15/7

z = 18/7

Given System

2x + y − z = 8

−3x − y + 2z = −11

−2x + y + 2z = −3

Step 1: Augmented Matrix

[ 2 1 -1 | 8 ]
[ -3 -1 2 | -11 ]
[ -2 1 2 | -3 ]

Step 2: Partial Pivoting

Compare absolute values in column 1: |2|, |-3|, and |-2|. Largest is |-3|, so swap Row 1 and Row 2.

R₁ ↔ R₂
[ -3 -1 2 | -11 ]
[ 2 1 -1 | 8 ]
[ -2 1 2 | -3 ]

Step 3: Eliminate Entries Below Pivot

m₂₁ = 2 / (-3) = -2/3

R₂ → R₂ − (-2/3)R₁

m₃₁ = (-2) / (-3) = 2/3

R₃ → R₃ − (2/3)R₁
[ -3 -1 2 | -11 ]
[ 0 1/3 1/3 | 2/3 ]
[ 0 5/3 2/3 | 13/3 ]

Step 4: Partial Pivoting (Column 2)

Compare absolute values in column 2: |1/3| and |5/3|. Largest is |5/3|, so swap Row 2 and Row 3.

R₂ ↔ R₃
[ -3 -1 2 | -11 ]
[ 0 5/3 2/3 | 13/3 ]
[ 0 1/3 1/3 | 2/3 ]

Step 5: Eliminate Below Second Pivot

m₃₂ = (1/3)/(5/3) = 1/5

R₃ → R₃ − (1/5)R₂
[ -3 -1 2 | -11 ]
[ 0 5/3 2/3 | 13/3 ]
[ 0 0 1/5 | -1/5 ]

Step 6: Back Substitution

z = (-1/5)/(1/5) = -1

(5/3)y + (2/3)(-1) = 13/3

y = 3

-3x - 3 + 2(-1) = -11

x = 2

Final Answer

x = 2

y = 3

z = -1
